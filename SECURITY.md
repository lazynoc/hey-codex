# Security

Hey Codex needs Microphone and Speech Recognition access for its on-device wake phrase, Accessibility access to trigger shortcuts and restore the cursor, and Automation access as a fallback if normal macOS app activation cannot restore focus. It does not need Full Disk Access.

Please use GitHub's [private vulnerability reporting](https://github.com/lazynoc/hey-codex/security/advisories/new) for security issues. Do not open a public issue or include private prompts, recordings, or machine information.

Security fixes are applied to the latest version on `main`; older source snapshots are not maintained separately.
